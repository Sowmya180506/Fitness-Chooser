# Fitness-Chooser
Personalized fitness app with mood, time, and workout-based YouTube recommendations.

This project helps users stay consistent with their fitness journey by intelligently filtering workouts from popular channels like Growingannanas, Caroline Girvan, Pamela Reif, and POPSUGAR Fitness.

Users can customize their experience by selecting:

⏱️ Workout duration (10–20 min, 20–40 min, 40+ min)
💪 Workout type (HIIT, Tabata, No Jumping, Strength, etc.)
😊 Mood-based workouts (Energetic, Relaxed, Feeling Low, etc.)

The app dynamically suggests relevant YouTube workouts, making it easier to find the right exercise for any day, time, or mindset.
